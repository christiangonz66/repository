The Colorado Job Tracker is a Streamlit application that visualizes job distribution across Colorado cities and counties.

Deployment Options

This application is designed to run on Streamlit Cloud. Vercel's serverless functions are not ideal for Streamlit apps.

Deploy on Streamlit Cloud

Instructions

  1. Clone the repository from GitHub
  2. Go to share.streamlit.io
  3. Connect your GitHub repository
  4. Set the main file path to app.py
  5. Click Deploy!

Local Development

To run this application locally:

pip install -r requirements.txt
streamlit run app.py
